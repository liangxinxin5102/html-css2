= Dusky =

Dusky is an elegant and responsive theme  with slider.  The theme can be used for any type of website : corporate, portfolio, business, blog,  etc. The theme displays excerpts, custom menus  and thumbnail images on the your blog.  
Slider shows sticky posts.
