Presentation

This is a free XHTML/CSS website template that you can use for any purpose without any obligations or limitations. It was originally released in August 2005, and has since been updated several times. It uses XHTML 1.0 Strict and CSS, and it has no tables at all. To show its features, I have filled it with some example content.
