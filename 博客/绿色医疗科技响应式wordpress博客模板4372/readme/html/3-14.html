广告设置

在后台“外观-AdminOptions-广告”中的设置。为一段广告代码取一个名称，好用于调用。

怎么使用？

在文章中使用短代码即可调用这个广告。例如，你增加了一个广告叫“边栏最顶部”，写文章的时候只需要使用[ad name="边栏最顶部"]即可调用这个广告对应的代码。

在前面已经提到过，广告的加载是异步加载的。

如何在主题中调用呢？使用下面的代码即可

global $admin_options;
$ads = array_admin_options($admin_options['ads']);
echo $google = stripslashes($ads['GOOGLE-300-250']);
// 获取的是广告代码
或
echo do_shortcode('[ad name="边栏最顶部"]');
// 广告代码将会异步加载
