社交媒体设置

在后台“外观-Admin Options-社交媒体”中设置。

设置后会在网页的右上角出现社交媒体按钮，用户点击后就会跳到对应的页面。

默认情况下，有几个灰色的是常用社交媒体账号。注意，右边的值一般都是链接，如果不是链接，你需要额外注意怎么使用，不能出现引号。

如何在前台调用？

你可以在前台使用PHP调用自己新增的社交媒体账号，代码如下：

global $admin_options;
$socials = array_admin_options($admin_options['social']);
$QQ = $socials['QQ'];

但是这仅仅是调用出它的值，css样式、按钮图片等还需要你自己添加。
