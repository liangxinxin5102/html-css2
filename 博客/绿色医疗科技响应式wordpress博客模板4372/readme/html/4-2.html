已有功能和方法

主题中已经包含了下面这些功能，你不需要再另外开发，只需要调用即可。

翻页导航

只需要在要显示翻页导航的loop结束wp_reset_posts()前面使用the_pagenavi();即可，当然，样式要你自己去调整。源码在includes/function-the_pagenavi.php中。

面包屑导航

只需要在显示当前位置的地方使用函数the_place('首页');即可，有一个参数，即“首页”链接内的文字是什么，默认情况下显示的是网站的名称。源码在includes/function-the_place.php中

获取文章缩略图，或以文章第一张图片作为缩略图

源码在includes/function-the_post_thumb.php中，获取文章的特色图片，如果不存在特色图片，则抓取文章第一张图片，如果文章不存在图片，就返回空。里面共有四个函数：

get_post_thumb_src($size = 'post-thumbnail',$default = false):获取缩略图的src地址
the_post_thumb_src($size = 'post-thumbnail',$default = false)：显示缩略图的地址
get_post_thumb($size = 'post-thumbnail',$default = false,$alt = false,$width = false,$height = false,$attrs = false)：获取缩略图的HTML代码
the_post_thumb($size = 'post-thumbnail',$default = false,$alt = false,$width = false,$height = false,$attrs = false)：显示图片

用户信息meta扩展

你可以增加用户在后台个人资料编辑中要填写的资料信息，打开inlcudes/user-metas.php，在你所看到的地方照葫芦画瓢，增加用户的信息就可以了。

把后台登陆界面的LOGO和链接更换为自己的

在主题目录images中放一个logo.png，即可替换这个LOGO。

文章被访问量统计

使用get_view_count($post_id = false)来获取该文章的被访问量，the_view_count($post_id = '')则是直接显示其数字。源码在includes/wp-post-view-count.php中。

获取后台admin options的设置值

使用get_admin_options($key = false)来获取某一个已知的设置值，在主题中直接声明global $admin_options;也可以。

$mode = get_admin_options('mode');// 即可获得当前的撰稿模式
或者
global $admin_options;
$mode = $admin_options['mode'];

使用update_admin_options($key,$value);来更新某值。

update_admin_options('mode',3);

使用array_admin_options($options = array());来数组化后台的社交媒体类的设置。

global $admin_options;
$socials = array_admin_options($admin_options['social']);
$QQ = $socials['QQ'];

获取admin options的配置信息

使用global $admin_options_config;来获取。
