欢迎使用Green Demo（下面简称GD）

Green Demo一款大气简洁的多功能高效主题，高度可自定义和内置的流行元素，帮助你快速搭建属于自己的网络平台。

我原本想把ReadMe文档做的比价酷炫一点，但是想想，毕竟这是一个让你懂得如何去操作这个主题的说明文档，应该直接切入主题才是，没有必要花哨和长篇大论，所以，现在开始吧。

Green Demo适合大部分普通用户的需求，我做它出来的主要想法是做自媒体类的网站，注重内容（我帮你解决了形式）。你可以发现它甚至具备了作为一个高端网站外壳的品质，只要你多下功夫，能够做到和雷锋、钛媒体之类的专业性网站的水准。

如果你：

  • 第一次使用WordPress（下称WP），那么从熟悉WP开始，你需要阅读第一章的所有内容。
  • 使用WP多年，直接进入第二章。第二章展示了GD的各种特性，你必须喜欢上它。
  • 对前台哪哪是怎么显示出来的搞不清楚，看第三章。
  • 打算在GD基础上做二次开发，必须看第四章。里面提到的Admin Options是我的得意之作，你肯定会在其他主题中用到。
